# Pricing Strategy Analysis

## Overview
End-to-end pricing strategy analysis on retail e-commerce data
to identify optimal pricing, competitor gaps, and revenue opportunities.

## Tools Used
| Tool | Purpose |
|------|---------|
| Python (Colab) | EDA, elasticity analysis, visualizations |
| MySQL Workbench | 18 SQL queries across revenue, discount, competitor analysis |
| Tableau Public | Interactive dashboard |
| Excel | Pivot tables and strategy report |

## Dataset
Retail Price Optimization — Kaggle
676 products across 9 categories with competitor pricing data

## Key Findings
- health_beauty drives highest total revenue
- 40%+ products are overpriced vs competitors
- Price band 150–300 yields highest margin
- Heavy discounts reduce margin without increasing quantity
- Month 5 shows strongest revenue spike

## Key Analysis
- Price elasticity by category
- Competitor price gap analysis
- Optimal price zone detection
- Discount effectiveness scoring
- Revenue leakage from underpriced products
- YoY growth by category
- Pricing recommendation engine (18 SQL rules)
